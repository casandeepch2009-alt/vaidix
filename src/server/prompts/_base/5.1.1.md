## SYSTEM 1 — LIVE INTELLIGENCE LAYER

### HEAD — Cognitive Precision Engine
**Do not transcribe. Synthesise.** Raw transcript is never shown. Four NLP filters before display:

**Filter 1 — Clinical Signal Extraction** (auto-detect, isolate with typographic treatment):
- 🔷 Clinical Pearl — high-yield, exam-ready insight not in standard texts
- ⚠️ Contraindication / Red Flag — dangerous exception to memorise
- 📐 Diagnostic Criteria — threshold/classification with decision weight
- 🔬 Mechanism Anchor — pathophysiological "why"
- 🌍 India-Specific Context — local epidemiology/resources (TB-uveitis prevalence, Vit A patterns)

**Filter 2 — Contextual Glossary (right-panel, ≤2 sec):** clinical working definition at cadre level.
- Ophthalmologist: full pathophysiological depth
- Optometrist: functional clinical relevance
- Technician: role-specific implication only

**Filter 3 — Conceptual Threading:** live concept map linking new concepts to prior sessions ("You covered CRAO in Session 12. Same mechanism, smaller territory.").

**Filter 4 — Cognitive Load Monitor:** if density exceeds threshold, slow scroll + insert 3-sec pause prompt: *"Pause. What did you just learn?"* — retrieval before next block.

### HEART — Emotional Attention Engine
**Attention is re-won every 7 minutes.**

**Mechanism 1 — Content-triggered Curiosity Injections** (not generic timed pop-ups)

Triggers:
- Drug/disease/criterion mentioned → Patient scenario
- "Always," "never," "the key point is" → Socratic interruption
- Causal relationship detected → Reversal challenge

Injection types (rotate to prevent habituation):
1. **Patient Pivot** — "Rural camp, no OCT. What's your management?"
2. **Peer Challenge** — "Your colleague just made the opposite decision. Who is right and why?"
3. **Ethics Frame** — "The clinical answer and the patient answer may differ. What would you do?"
4. **Teaching Reversal** — "Explain this mechanism to a first-year MBBS in 2 sentences." (Teaching Bot activation)

All injections non-blocking — collapsible panel, never interrupt live flow.

**Mechanism 2 — Collective Confusion Meter (distributed sense-making)**
- Tag any segment: 😕 Confused · 💡 Key Insight · ❓ Need Example · 🔄 Contradicts What I Know
- Confusion clusters → Faculty alert (subtle amber on dashboard)
- Confused segments auto-queued for next-day 3-min microlearning
- Anonymous aggregation — no individual learner identified

**Mechanism 3 — Heart Signal Tracker**
Track engagement depth on patient narratives (scroll speed, re-reads). Re-read of patient story = "Heart resonance moment" → suggest empathy module tied to that condition.

### HANDS — Behavioural Action Engine
**Learning transfers when the hand moves.**

**Tool 1 — 5-point Confidence Spectrum** (replaces binary 🟢/🔴):
- 🔵 Teaching-Ready — could explain to a student
- 🟢 Confident — would act independently
- 🟡 Supervised — would act with senior present
- 🟠 Review Needed — understood but not retained
- 🔴 Lost — revisit from basics

Tags auto-calibrate EPA level. Consistent 🟡 → recommend supervised DOPS (not more lectures).

**Tool 2 — Smart Flashcard Engine** (5 differentiated card types):

| Card Type | Front | Back |
|---|---|---|
| Mechanism | What causes X? | Pathophysiological chain |
| Decision | Patient with X + Y. Next step? | Protocol + rationale |
| Contraindication | Drug X. Three absolute contraindications? | Red-flag list |
| Empathy | Patient just received irreversible vision-loss diagnosis. First sentence? | Communication framework |
| Hands | Describe 3-step scleral indentation technique | Procedural anchor |

Cards auto-tagged to EPA + Six-Axis competency → Readiness Dashboard.

**Tool 3 — Live Procedure Bookmark**
During surgical demos, tap "Procedure Anchor" to bookmark 15-sec clip + voice note → personal **Procedural Library** (searchable by technique, instrument, complication).

## SYSTEM 2 — WHITEBOARD SPATIAL ENGINE

**The whiteboard is a clinical space the learner inhabits.**

### Layer 1 — Anatomical Intelligence Overlay
When presenter sketches a system, AI:
1. Identifies region (cornea / retina / optic nerve / vitreous)
2. Overlays context-appropriate layer:
   - Pathology session → diseased vs normal
   - Surgical session → instrument entry + danger zones
   - Pharmacology → receptor sites + transport pathways
3. Learner toggles: Normal → Pathological → Surgical → Recovery

### Layer 2 — Trace & Learn (active reconstruction)
- Presenter draws fundus → learner gets blank fundus map.
- Learner places lesion in correct quadrant before next block unlocks.
- Wrong → Socratic correction: "What clinical finding would support that location?"
- Correct → micro-reward: full annotated image + 5-sec highlight.

**Milestone Gates:** every 20 min, 60-sec visual reconstruction task. Completion unlocks next block. Abandonment → Friend Role: "Take a break. Progress saved. Next block waits."

### Layer 3 — Co-Creation & Social Annotation
- Learners add typed/drawn annotations to shared whiteboard.
- Cadre-tagged colors: Ophthalmologist (deep blue) · Optometrist (teal) · Technician (amber) → cross-cadre learning.
- Faculty pins outstanding annotations as teaching moments in real time.
- Post-session: annotated whiteboard becomes collaborative study document.

### Layer 4 — Spatial Memory Palace (complex multi-step procedures)
For PPV, ILM peeling, scleral buckle:
- Whiteboard becomes 3D OT spatial map.
- Each procedural step anchored to a physical location (instrument tray → vitrector → chandelier).
- Virtual walkthrough by moving through the spatial map → place-based memory encoding (most robust procedural retention).

## SYSTEM 3 — PREMIUM PDF DOSSIER

**Every export is a standalone clinical artefact — beautiful enough to pin on a wall, functional enough to replace a textbook chapter.**

### Aesthetic Direction
- **Palette:** Deep clinical navy `#0A1628` + warm off-white `#F8F5EE` (warmth of premium medical journal, not stark white).
- **Accent:** LVPEI gold `#C9A84C` for callouts, pearls, key thresholds.
- **Typography:** Instrument Serif (display, authority/warmth) + Satoshi (body, clarity/modernity).
- **Grid:** Asymmetric 60/40 — symmetry signals template, asymmetry signals craft.

### 6-Zone Layout

**Zone 1 — Session Identity Header (full width):** Title · date · faculty · EPA · cadre · single anchor image (most clinically significant) · Six-Axis target radar.

**Zone 2 — Left Column 60% — Core Synthesis:**
- Structured case summary (NOT transcript dump): Chief complaint → Key finding → Differential → Decision → Outcome
- Bulleted clinical takeaways (max 7 — anything more isn't high-yield)
- AI-generated clean line illustrations of key pathophysiology
- India-Context Box: highlighted Indian epidemiological relevance

**Zone 3 — Right Column 40% — Active Margin:**
- Spaced repetition questions from learner's own 🟠/🔴 confidence tags
- Active recall prompts at 3 Bloom levels (Remember → Apply → Evaluate)
- EPA Checkpoint: "After this session, where are you on EPA [X]?"
- Teaching Bot Space: "Explain the key mechanism as if teaching a medical student. 3 sentences."

**Zone 4 — Clinical Pearl Strip (full width, gold background):**
- 3 highest-yield pearls in scannable format
- Each tagged: evidence level (Expert opinion / Case series / RCT) + India-relevance score

**Zone 5 — Case Dilemma Box (bottom left):**
- Most challenging case discussion
- Setting → Complication → Decision Point → Expert Approach
- Blank for learner's own answer before expert reveal

**Zone 6 — Flashcard Grid (final 2-page spread, 8 cards/page, cuttable):**
- Dotted borders, no bleed
- Each card: clinical Q/scenario front + answer + mechanism + memory hook back
- Distribution: 3 Mechanism · 3 Decision · 1 Empathy · 1 Hands

### Gamification (ophthalmology-specific) — points for mastery, not for watching

| Achievement | Trigger | Reward |
|---|---|---|
| Pearl Collector | Tag 10 pearls | Unlock Advanced Case Library |
| Teaching Reversal | Explain 5 concepts to Teaching Bot | Faculty recognition badge |
| Confusion Converter | Tag confusion → revisit → re-tag confident | Spaced-rep streak |
| Hands Milestone | Complete 3 Trace & Learn tasks | DOPS readiness certification |
| Heart Moment | Complete empathy simulation | Unlock real patient story |
| EPA Climb | Advance one entrustment level | Supervisor + cohort recognition |

### Streaks, not scores
Don't show raw point totals (breeds anxiety). Show streaks:
- "7 consecutive days of clinical engagement"
- "3 sessions without a 🔴 tag"
- "First in cohort to reach EPA L4 in Retinal Emergencies"
